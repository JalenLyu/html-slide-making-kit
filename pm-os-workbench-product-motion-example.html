按钮先明确读什么

Tool

再明确调用什么

Output

最后明确写哪里

输出回到 Reports、Candidates 或 Projects，PM 决定是否写回事实源。

1
Source从 Inbox、Calendar、Email、AI Hot、Jira 或项目卡读取上下文。

2
Tool用 MCP、CLI 或 Skill 执行摘要、归类、检查、更新。
run
3
AgentCodex 根据资料和规则生成候选结果。
draft
4
Output结果写到 Reports、Candidates 或 Projects。
write
5
ConfirmPM 在 Workbench 判断是否采纳或继续处理。
decide
